# 🫧 MoodBubble

A map-based emotional journaling platform — record your feelings at specific locations and visualize your emotional landscape over time.

Built for **UNIHACK** hackathon.

## Quick Start

```bash
npm install
cp .env.example .env.local   # fill in your keys
npm run dev                   # http://localhost:3000
```

## Tech Stack

- **Next.js 16** (App Router) + **TypeScript**
- **Supabase** (PostgreSQL, Auth, Storage)
- **React Leaflet** + OpenStreetMap (interactive map, free)
- **Tailwind CSS v4**
- **Vercel** (deployment)
